== Description ==

*Update the wordpress login logo, logo link and logo title on hover

Usage :

1. Download and extract this plugin to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. "Dashboard"->"Settings"->"Update WP login logo"
